# -Detecci-n-en-tiempo-real-de-ataques-DDoS-mediante-an-lisis-de-tr-fico-con-Suricata-y-Grafana
Simular ataques de denegación de servicio (DDoS) en una red virtualizada con Mininet y monitorizar el tráfico en tiempo real con Suricata. Visualizar alertas y estadísticas en Grafana. Usamos: Mininet, Suricata, Grafana, Python, Wireshark.
