# Detección en tiempo real de ataques DDoS mediante análisis de tráfico con Suricata y Grafana
- Simular ataques de denegación de servicio (DDoS) en una red virtualizada con Mininet, y monitorizar el tráfico en tiempo real con Suricata.
- Visualizar alertas y estadísticas en Grafana.

# Descripción:
En este proyecto se simularán ataques de denegación de servicio distribuida (DDoS) sobre una red virtualizada con Mininet, generando distintos patrones de tráfico malicioso y legítimo. El tráfico será monitorizado en tiempo real con Suricata, que se encargará de la inspección profunda de paquetes y la generación de alertas de seguridad.
Las métricas y eventos detectados se enviarán a una base de datos de series temporales y se visualizarán en paneles de Grafana, permitiendo analizar el comportamiento del ataque (volumen, origen, evolución temporal) y la eficacia de las reglas de detección. Además, se discutirán el impacto energético de los ataques DDoS sobre la infraestructura y cómo una detección temprana y eficiente puede contribuir a reducir consumo, sobredimensionamiento y residuos electrónicos asociados.
